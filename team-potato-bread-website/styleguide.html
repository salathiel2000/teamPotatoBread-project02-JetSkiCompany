TEXT STYLING

The website uses Monsterrat at varying sizes and weights. These variations allow for hierarchy to be established throughout the website.

Headings

Main Heading

2.25 rem

Bold

Main Headings are used only once on the page, at the top to give the user evidence of the page they have arrived to. They appear at top in yellow in front of a banner image.

Sample code

HTML

<h1>This is the main heading</h1>

CSS

h1{
font-size: 2.25rem;
font-weight: bold;
color: #e9af00;
}

Sub-heading

1.75 rem

Semibold

Sub-headings are used as any main heading on the page that come beneath the banner image that has the main heading

Sample code

HTML

<h2>This is a sub-heading</h2>

CSS

h2{
font-size: 1.75rem;
font-weight: 600;
}

INTERACTIVE ELEMENTS

The navigation is at the top right of the page. All navigation elements are exposed to allow the user to easily access the page they want. They encompass the overall brand of the page with the navy-blue background and gold type. There's a call-to-action by the particular navigation button being underlined on hover.

FORM ELEMENTS

Forms are used throughout the website for its primary goal, which is to book a jet ski rental. They are also used to collect data to provide a suggestion to users who've never rented or ridden a jet ski before

Input field

Input fields are used within the Booking Section to allow the user to enter their data for the booking

#lname
